# CV Bullets — Decision Tree Classification Project

Use one or two bullets depending on available CV space.

## Recommended English bullets

- Built an interpretable Decision Tree classification demo using Python, scikit-learn, and Streamlit, including prediction probabilities, decision-path explanation, confusion matrix, and classification report.

- Audited a legacy machine learning notebook, identified target leakage behind a misleading perfect accuracy result, and reframed the project into a responsible model evaluation and explainability case study.

- Developed a scikit-learn preprocessing pipeline for tabular classification with missing-value handling, one-hot encoding, stratified train/test split, and DecisionTreeClassifier evaluation.

- Created a recruiter-facing ML portfolio demo focused on classification, explainability, and visualization using Streamlit and Decision Tree rule-path interpretation.

## Recommended Indonesian bullets

- Membuat demo klasifikasi Decision Tree berbasis Python, scikit-learn, dan Streamlit dengan prediksi class, probabilitas, rule-path explanation, confusion matrix, dan classification report.

- Mengaudit notebook machine learning lama, menemukan indikasi target leakage di balik hasil akurasi sempurna, lalu menyusun ulang project menjadi studi kasus evaluasi model dan explainability.

- Membangun pipeline preprocessing scikit-learn untuk data tabular dengan imputasi missing value, one-hot encoding, stratified train/test split, dan evaluasi DecisionTreeClassifier.

- Menyiapkan demo portfolio machine learning untuk recruiter dengan fokus pada classification, explainability, evaluation, dan visualization.

## Short LinkedIn version

- Turned a legacy Decision Tree notebook into a recruiter-facing interpretability demo with Streamlit, rule-path explanations, confusion matrix, and classification report.

## Avoid these claims

Do not write:

```text
Achieved 100% accuracy in income prediction.
```

Do not write:

```text
Built a production-ready income prediction system.
```

Do not write:

```text
Created a fair automated income decision model.
```

## Safer wording

Use:

```text
Portfolio demo
Educational classification project
Decision Tree interpretability demo
Model evaluation case study
Legacy notebook audit and refactor
```
